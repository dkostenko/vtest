# HTTPSERV

Сервис предоставляет HTTP API для работы с постами / подписками.
Сервис состоит из двух основных частей:
- пакет работы с HTTP запросами
- пакет работы с БД

## Установка

```
$ composer install
```

## Запуск

```
DB_ADDR_URI='tcp://127.0.0.1:3301' \
DB_USER_NAME=guest \
DB_USER_PASSWORD='' \
php -S  0.0.0.0:8080 -t public/
```
